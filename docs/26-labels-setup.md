# Настройка меток (labels)

Как создать и настроить метки в репозитории на GitHub.

## Цель

- Добавить рекомендуемый набор меток (см. [Справочник меток](25-labels-reference.md)).
- Задать цвет и описание для каждой.

## Шаги

1. В репозитории: **Issues** → **Labels** (или Settings → в боковой панели Labels).
2. Нажми **New label**.
3. Введи имя (например `docs`), выбери цвет, при желании добавь описание.
4. Повтори для всех меток из справочника.

## Массовое создание

- Вручную по одной — самый простой способ.
- Через GitHub API или скрипт — при большом наборе (для этого репо достаточно вручную).

## Проверка

- На странице Labels отображаются все созданные метки.
- При создании issue в выпадающем списке Labels видны новые метки.
