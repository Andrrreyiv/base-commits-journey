# Создание и клонирование репозитория

Как создать публичный репозиторий на GitHub и склонировать его по HTTPS.

## Цель

- Публичный репозиторий создан на GitHub.
- Репозиторий склонирован на компьютер (clone по HTTPS).

## Шаги

1. Зайди на [github.com](https://github.com), войди в аккаунт.
2. Кнопка **New** (или «+» → New repository).
3. Укажи имя репозитория, выбери **Public**, при желании добавь README (можно пустой).
4. Нажми **Create repository**.
5. На странице репозитория нажми **Code**, скопируй URL по HTTPS (вид: `https://github.com/username/repo-name.git`).
6. В терминале перейди в папку, где хочешь держать проекты, и выполни:
   ```bash
   git clone https://github.com/username/repo-name.git
   cd repo-name
   ```

## Проверка

- В папке `repo-name` есть скрытая папка `.git` и файлы репозитория.
- `git status` внутри папки выполняется без ошибок.

## Типовые ошибки

- **400 Bad Request:** проверь URL (без лишних пробелов, с `.git` в конце).
- **404 Not Found:** репозиторий не существует или приватный, а у тебя нет доступа; проверь имя и владельца.
- **Authentication failed:** нужна настройка входа (см. гайд [Аутентификация по HTTPS](05-auth-https-gcm.md)).
