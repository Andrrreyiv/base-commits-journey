# Рекомендуемые метки (labels)

Справочник меток для issue и PR в этом репозитории.

## Цель

- Единый набор меток для фильтрации и приоритета.
- Быстро находить задачи по типу.

## Рекомендуемый набор

| Метка | Цвет | Назначение |
|-------|------|------------|
| bug | красный | Ошибка в коде или документации |
| enhancement | синий | Новая возможность, улучшение |
| docs | зелёный | Только документация |
| chore | серый | Настройки, конфиги, без логики |
| ci | жёлтый | CI/CD, workflow, скрипты |
| good first issue | зелёный | Подходит для новичков |
| help wanted | оранжевый | Нужна помощь сообщества |
| priority: high | красный | Высокий приоритет |
| priority: low | серый | Низкий приоритет |

## Использование

- При создании issue выбери одну или несколько меток.
- В PR можно оставить те же метки или добавить по типу изменений.
- Фильтр по меткам: на странице Issues выбери Label.
