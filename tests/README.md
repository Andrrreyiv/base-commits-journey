# Тесты

Юнит-тесты для скриптов в `scripts/`. Только stdlib (unittest).

## Запуск

Из корня репозитория:

```bash
python -m unittest discover -v -s tests
```

Или через Makefile (если есть цель `test`).

## Структура

- `test_*.py` — модули с тестами (по одному на скрипт или группу).
- `fixtures/` — тестовые файлы (markdown, конфиги) для проверки скриптов.

## Правила

- Без внешних зависимостей (только Python stdlib).
- Тесты не меняют файлы в корне репо (используют fixtures или временные каталоги).
