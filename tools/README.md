# Tools

Вспомогательные CLI для работы с репозиторием.

## base_commit_coach.py

Подсказки по прогрессу и следующему шагу.

```bash
python tools/base_commit_coach.py --help
python tools/base_commit_coach.py --report   # число коммитов, записей в PROGRESS
python tools/base_commit_coach.py --next     # подсказка: следующая задача
```

Только Python stdlib.
